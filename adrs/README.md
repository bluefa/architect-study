# ADRs Guide

주차 Drill에서 확정한 의사결정을 요약/추적합니다.

## 규칙
- 주차 폴더에 원본 ADR을 저장: `drills/Wxx-topic/05-adr-00n.md`
- `adrs/`는 인덱스나 요약 문서 용도로 사용
- ADR은 반드시 Context/Decision/Consequences/Alternatives/Notes 구조를 따른다.

| ADR | 주차 | 결정 | 상태 |
|---|---|---|---|
| ADR-001 | W01 | 예: 키 발급 전략 | Accepted |
