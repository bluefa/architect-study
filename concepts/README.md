# Concepts Guide

주차 Drill에서 배운 핵심 개념을 카드 형태로 누적합니다.

## 작성 규칙
- 파일명: `concepts/YYYY-MM-DD-topic-concept.md`
- 1개 카드 = 1개 개념 원칙
- 10분 안에 읽히는 길이로 작성
- 정의보다 "언제 쓰고 언제 버리는지"를 우선 기록
- 각 카드에 최소 1개 반례 또는 실패 사례를 포함

## 최소 필드
- 개념 정의
- 적용 조건
- 장점/한계
- 실패 신호
- 관련 Drill/ADR 연결 경로

## 권장 워크플로
1. 주차 리뷰 직후 개념 후보 3개를 적는다.
2. 그중 재사용 가치가 높은 1~2개를 카드로 확정한다.
3. 다음 주 설계 전에 카드를 재검토한다.
